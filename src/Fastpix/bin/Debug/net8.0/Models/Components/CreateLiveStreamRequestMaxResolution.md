# CreateLiveStreamRequestMaxResolution

Defines the maximum resolution for encoding, storage, and playback of the live stream.



## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `OneThousandAndEightyp`  | 1080p                    |
| `SevenHundredAndTwentyp` | 720p                     |
| `FourHundredAndEightyp`  | 480p                     |
